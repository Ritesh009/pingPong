# Pong Game Documentation

## Introduction
Pong is one of the earliest arcade video games and is a simple two-dimensional sports game that simulates table tennis. Players control paddles to hit a ball back and forth.

## How to Play
- Each player controls a paddle by dragging it vertically across the screen's left or right side.
- The objective of the game is to score points by getting the ball past the opponent's paddle.
- Players can choose to play against each other or against the computer.

## Game Features
- Simple controls
- Two-player mode
- Computer AI for single player
